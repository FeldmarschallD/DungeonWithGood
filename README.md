# DungeonWithGood
A web application that allows DMs to run DnD games flexibly

Usage
- Open the HTML file
- Keyboard controls:
- - K - load background image and/or audio
- - O - load character sprite
- - I - toggle sprite visibility
- - M - toggle auido-player visibility
- - Space - play/pause audio
- - Arrow Left/Right - seek 5 seconds
- - Arrow Up/Down - volume control
- - L - toggle loop mode
- - D - open/close dice roller

This is a hobby project created for personal use and shared as-is. It contains known limitations and unfinished features. The codebase requires refinement and expansion.
